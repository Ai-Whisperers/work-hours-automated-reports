## Quick Reference

**Base URL**: `https://api.clockify.me/api/v1`
**Rate Limit**: 50 requests/second per workspace
**Authentication**: `X-Api-Key` header

## Regional Servers

- EU: `https://api.clockify.me`
- USA: `https://api.clockify.me`
- UK: `https://api.clockify.me`
- AU: `https://api.clockify.me`

## Key Concepts

- **Workspace**: Container for time tracking data
- **User**: Team member with access to workspaces
- **Project**: Time tracking category
- **Time Entry**: Logged time record
- **Tag**: Label for categorizing time entries
- **Client**: Customer associated with projects
